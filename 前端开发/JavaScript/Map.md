# Map 的属性和方法

<aside>
📌 数据结构 Map 也实现了 Iterator 接口，所以可以使用【扩展运算符】或【for…of】进行遍历。

</aside>

| 属性或方法名 | 描述 |
| --- | --- |
| `set(key, value)` | 增加一个新的元素，返回当前 Map 。 |
| `delete(key)` | 删除 Map 中的某个元素。 |
| `get(key)` | 返回键名对象的键值。 |
| `clear()` | 清空 Map ，返回 undefined。 |
| `size` | 返回 Map 的元素个数。 |
| `has` | 检测 Map 中是否包含某个元素，返回布尔值。 |
